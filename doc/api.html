API

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-state="light". 

<div id="api-example-1">
    <button class="btn btn-primary" data-plugin="bsdarkmode" data-root="#api-example-1" data-state="dark" data-lightvars='{"primary":"#FF0000","secondary":"#00FF00"}' data-darkvars='{"primary":"#0000FF","secondary":"#FFFF00"}'></button>
</div>
<div id="api-example-2">
    <button class="btn btn-primary" id="darkmodeToggle"></button>
</div>
<script>
$(function() {
$('#darkmodeToggle').bsDarkmode({
    state: false,
    root:'#api-example-2',
    lightvars: '{"primary":"#FF0000","secondary":"#00FF00"}',
    darkvars: '{"primary":"#0000FF","secondary":"#FFFF00"}'
});
})
</script>

Bootstrap Darkmode API options
Name Type Default Description
state boolean true Initial state (For data-state use light or dark).
The user preferred color scheme dark is prioritary.
root string ":root" Root element to apply CSS vars
lightlabel html sun svg Element inner HTML for light mode
darklabel html moon svg Element inner HTML for dark mode
lightvars string "{}" JSON object with CSS vars for light mode
darkvars string "{}" JSON object with CSS vars for dark mode

Methods

Methods can be used to control toggles directly.

Bootstrap Darkmode API methods description
Method Example Description Demo
initialize $('#darkmodeMethods').bsDarkmode() Initializes the darkmode plugin with options
light $('#darkmodeMethods').bsDarkmode('light') Sets the darkmode toggle to 'light' state
dark $('#darkmodeMethods').bsDarkmode('dark') Sets the darkmode toggle to 'dark' state
toggle $('#darkmodeMethods').bsDarkmode('toggle') Toggles the state of the darkmode toggle light/dark
For ECMAS JavaScript replace $('#toggle-demo')... with document.querySelector('#toggle-demo')...
