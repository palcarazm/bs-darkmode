Features

State

Bootstrap Darkmode initial state can be set by data-state attribute.
The user preferred color scheme dark is prioritary.

Custom Root Element

Bootstrap Darkmode root element can be set by data-root attribute.

Custom Label

Bootstrap Darkmode light and dark label can be set by data-lightlabel and data-darklabel attributes.

Custom Theme

Bootstrap Darkmode css Theme can be set by data-lightvars and data-darkvars attributes. Vars are defined in a JSON object.

<div id="theme-example">
    <button class="btn btn-primary" data-plugin="bsdarkmode" data-lightlabel="LIGHTS ON" data-darklabel="LIGHTS OFF" data-root="#theme-example" data-lightvars='{"primary":"#FF0000","secondary":"#00FF00"}' data-darkvars='{"primary":"#0000FF","secondary":"#FFFF00"}'></button>
</div>

Prefered Scheme in Cookie

Bootstrap Darkmode can save user prefered color schema in a cookie. To enable this feature add allowsCookie attribute.

As per the GDPR and ePrivacy Directive, a website must ask its users’ consent to use cookies that are not necessary for accessing the website’s functionality. These cookies need consent because they collect user data for their purposes. According to the law, collecting data without users’ consent is unlawful.

The employed cookie is a User interface (UI) customization cookie that do not requiere a explicit authorization. However the default value is set to false in order to allow requesting cookie consent if wanted.

If consent is provided, the cookie usage can be enabled/disabled with setCookieAutorization method.
