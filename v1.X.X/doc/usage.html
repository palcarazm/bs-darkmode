Usage

Initialize with HTML

Simply add data-toggle="toggle" to convert checkboxes into toggles.

<button class="btn btn-primary" data-plugin="bsdarkmode"></button>

Initialize with JavaScript

Simply call the bootstrapToggle method to convert checkboxes into toggles. See Options for additional colors, etc.

$('[data-plugin="bsdarkmode"]').bsDarkmode()
document.querySelectorAll('[data-plugin="bsdarkmode"]').forEach(function(ele) {
    ele.bsDarkmode();
});
